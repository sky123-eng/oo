# 无障碍功能使用说明

## 简介

我们致力于为所有用户提供平等、友好的使用体验。本应用提供了多种无障碍功能，以满足不同能力用户的需求。本指南将帮助您了解如何开启和使用这些功能。

## 如何访问无障碍控制面板

1. **打开控制面板**：
   - 在应用界面中，寻找并点击「无障碍设置」按钮（通常显示为齿轮图标）
   - 或使用快捷键 `Ctrl + Alt + A`（Windows）或 `Command + Option + A`（Mac）快速打开

2. **控制面板布局**：
   - 控制面板包含多个功能开关和调整选项
   - 所有控件都支持键盘导航（使用Tab键切换，Enter或Space键激活）
   - 每个控件都有清晰的标签和说明文字

## 主要无障碍功能

### 1. 高对比度模式

**功能介绍**：增强界面元素的颜色对比度，使文字和图形更容易阅读。

**使用方法**：
1. 在无障碍控制面板中，找到「高对比度模式」开关
2. 点击开关或使用键盘激活，开启/关闭高对比度模式

**效果**：
- 文字与背景的对比度增强
- 主要操作按钮更加突出
- 界面元素边框加粗
- 颜色方案调整为更易辨识的组合

### 2. 字体大小调整

**功能介绍**：允许您调整应用中的文字大小，适应不同的视力需求。

**使用方法**：
1. 在无障碍控制面板中，找到「字体大小」滑块或选项
2. 拖动滑块或选择预设大小（小、标准、大、超大）
3. 实时预览调整效果

**可用选项**：
- 小：适合屏幕空间有限或偏好紧凑布局的用户
- 标准：默认字体大小
- 大：增大字体，提高可读性
- 超大：最大字体选项，适合严重视力障碍用户

### 3. 语音导航

**功能介绍**：提供语音反馈和操作引导，帮助视觉障碍用户了解界面和操作结果。

**使用方法**：
1. 在无障碍控制面板中，找到「语音导航」开关
2. 点击开关或使用键盘激活，开启/关闭语音导航

**功能特点**：
- 操作反馈：执行操作时提供语音确认
- 状态提示：重要状态变化时的语音通知
- 错误提示：操作错误时的语音警告
- 可调节音量：根据需要调整语音提示音量

### 4. 单手操作布局

**功能介绍**：简化界面布局，优化单手操作体验，特别适合行动不便或使用移动设备的用户。

**使用方法**：
1. 在无障碍控制面板中，找到「单手操作模式」开关
2. 点击开关或使用键盘激活，切换到单手操作布局
3. 要退出单手操作模式，再次点击开关或使用界面中的「退出单手模式」按钮

**布局特点**：
- 垂直排列的简化界面
- 超大触摸按钮（≥48px），易于点击
- 底部固定导航栏，方便单手触及
- 核心功能快速访问
- 减少操作步骤，简化导航路径

### 5. 简化布局

**功能介绍**：简化界面元素和导航结构，减少视觉干扰，提高重点内容的可见性。

**使用方法**：
1. 在无障碍控制面板中，找到「简化布局」开关
2. 点击开关或使用键盘激活，开启/关闭简化布局

**效果**：
- 减少非必要的装饰元素
- 简化导航菜单结构
- 增大内容区域比例
- 突出显示核心功能和信息

## 键盘导航指南

本应用支持完全的键盘导航，确保所有功能可通过键盘访问和操作：

### 基本导航键

- **Tab**：在界面元素之间向前移动焦点
- **Shift + Tab**：在界面元素之间向后移动焦点
- **Enter**：激活当前选中的按钮或链接
- **Space**：激活当前选中的按钮或切换开关状态
- **Arrow Keys**：在列表、菜单或滑块中移动选择
- **Esc**：关闭弹窗、下拉菜单或退出当前模式

### 快捷键

- **Ctrl + Alt + A**：快速打开/关闭无障碍控制面板
- **Ctrl + +**：增大字体大小（在支持的浏览器中）
- **Ctrl + -**：减小字体大小（在支持的浏览器中）
- **Ctrl + 0**：恢复默认字体大小（在支持的浏览器中）

## 常见问题解答

### Q: 开启高对比度模式后，某些图片或图标显示异常怎么办？
A: 高对比度模式可能会影响某些图片的显示效果。您可以尝试关闭高对比度模式，或联系我们报告问题图片，我们将进行优化。

### Q: 字体大小调整后，部分界面元素显示不完整怎么办？
A: 应用设计支持自适应布局，但在极端字体大小下可能出现布局问题。建议尝试使用稍小一级的字体大小，或联系我们报告具体问题。

### Q: 语音导航与我的屏幕阅读器冲突怎么办？
A: 您可以选择关闭应用的语音导航功能，仅使用屏幕阅读器，或者调整两者的音量以减少干扰。

### Q: 单手操作模式下，如何访问完整功能？
A: 单手操作模式提供了核心功能的快速访问。要使用完整功能，您可以暂时退出单手操作模式，或通过侧边菜单访问高级功能。

### Q: 如何保存我的无障碍设置？
A: 应用会自动保存您的无障碍设置，下次打开应用时将自动应用这些设置。
